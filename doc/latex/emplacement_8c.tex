Implémentations des fonctions pour la gestion des emplacements dans le jeu. 

Ce fichier contient les implémentations des fonctions pour créer, vérifier, ajouter, rechercher, afficher, libérer, et déterminer le nombre d\textquotesingle{}emplacements, ainsi que pour vérifier si une position est un emplacement.

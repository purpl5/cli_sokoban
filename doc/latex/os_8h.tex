\doxysubsection{Description détaillée}
Définitions pour la gestion de la compatibilité entre différents systèmes d\textquotesingle{}exploitation. 

Ce fichier contient des directives de préprocesseur pour déterminer l\textquotesingle{}environnement d\textquotesingle{}exécution du jeu (Windows, Unix, etc.) et définir des constantes en conséquence. Il est utilisé pour gérer les différences entre les systèmes d\textquotesingle{}exploitation dans le code.

\doxysubsection{Description détaillée}
Structure représentant l\textquotesingle{}état global du jeu. 

Cette structure contient tous les éléments nécessaires pour gérer l\textquotesingle{}état du jeu, y compris le niveau actuel, les emplacements ou il faut mettre les blocs, et les informations sur le joueur.

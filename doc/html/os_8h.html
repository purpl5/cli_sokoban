Description détaillée

Définitions pour la gestion de la compatibilité entre différents systèmes d'exploitation.

Ce fichier contient des directives de préprocesseur pour déterminer l'environnement d'exécution du jeu (Windows, Unix, etc.) et définir des constantes en conséquence. Il est utilisé pour gérer les différences entre les systèmes d'exploitation dans le code.

Description détaillée

Définitions et déclarations pour la manipulation des blocs dans le jeu.

Ce fichier contient les déclarations de fonctions nécessaires pour gérer le déplacement et la manipulation des blocs dans les niveaux du jeu.

Documentation des fonctions

◆ deplacementBlocX()

void deplacementBlocX ( Level *  l,
FileEmplacement *  f,
Position *  pl,
int  sens,
int *  nbBlocSurEmplacement 
)

Déplace un bloc dans le niveau sur l'axe X.

Cette fonction modifie la position d'un bloc dans le level (représenté par 'l') en déplaçant le bloc le long de l'axe X en fonction du paramètre 'sens'.

Paramètres
lPointeur vers le level dans lequel le bloc est déplacé.
fPointeur vers la file des emplacements.
plPointeur vers la position actuelle du bloc.
sensDirection du déplacement. Les valeurs positives indiquent un déplacement vers le bas, et les valeurs négatives vers le haut.
nbBlocSurEmplacementPointeur vers un entier qui est mis à jour pour refléter le nombre de blocs sur les emplacement après le déplacement du bloc.
